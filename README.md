# MusicScraperOrganiser
Downloads music with yt-dlp and organises it.

# Installation
This project requires Python 3 and the `mutagen` library:

```bash
pip install mutagen
```

## For yt-dlp
## Linux
```bash
curl -L https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp -o yt-dlp
chmod +x yt-dlp
mkdir -p ~/.local/bin/
mv yt-dlp ~/.local/bin 
```
